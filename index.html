Design

Project Dodge: a lightweight game for PC and mobile using Unity.

The average human reaction time to visual stimulus is ~260ms. Audio is slightly faster at ~170ms. With training this speed can approach ~150ms but has an absolute limit of ~110ms. This game features several difficulties which will determine warning time and projectile speed. I will also be attempting to create an intelligent AI that will try and detect patterns in the player's movements and start predicting them.
